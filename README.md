## Purpose
This is a sandbox for experimenting with different component designs and styles. Each component can be copied and used in other projects, with JavaScript added later for full functionality.

## CSS Techniques Used
- `border-radius` — rounded and pill shapes
- `::placeholder` — styling input hint text
- `backdrop-filter: blur()` — glass effect (glassmorphism)
- `rgba()` — colours with transparency
- `linear-gradient` — colourful backgrounds
- `transition` — smooth hover effects
- `:focus` — interactive styling
- `box-shadow` — depth and elevation
