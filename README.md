# Component Library

A collection of reusable UI components for web projects.

## Components

### Search Boxes
- Search Box 1: Minimal & Clean
- Search Box 2: Bold & Colorful 🌈
- Search Box 3: Rounded & Soft ⭕
- Search Box 4: Dark Mode 🌙
- Search Box 5: Glass Effect ✨

## Purpose
This is a sandbox for experimenting with different component designs
and styles. Each component can be copied and used in other projects,
with JavaScript added later for full functionality.

## CSS Techniques Used
- `border-radius` — rounded and pill shapes
- `::placeholder` — styling input hint text
- `backdrop-filter: blur()` — glass effect
- `rgba()` — colours with transparency
- `linear-gradient` — colourful backgrounds
- `transition` — smooth hover effects
